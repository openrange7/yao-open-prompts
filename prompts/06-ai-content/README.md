# AI内容目录

本目录存放写作、标题、公众号 HTML、短视频、内容运营、图像、PPT 创意等内容生产类提示词。

新增的 36 个内容与运营提示词已直接放在本级目录下，采用与仓库其他提示词一致的英文文件名。每个文件保留独立 Prompt 正文、开场引导语和示例输入输出，便于单独复制、测试和迭代。

## 36 个内容与运营提示词

这批提示词按实际使用场景重新归为六组：文案创作、人设风格、内容工具、平台运营、行业内容、运营辅助。

## 使用建议

- 内容创作和短视频场景优先从文案创作、内容工具、平台运营三组开始。
- 垂直行业账号可直接使用行业内容组作为长期选题和脚本生产模板。
- 直播转化、私域沟通、评论区运营、数据复盘等场景使用运营辅助组，并根据平台规则调整表达边界。
